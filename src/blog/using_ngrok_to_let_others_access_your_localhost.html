Using ngrok to let others access your localhost

You can easily use ngrok to host your localhost into their public host for free.

The main reason I do this is for the times when I wanna show my website to someone that is not on the same network as me. The great news is that you can use ngrok for both your static or dynamic website.

How to do it?

  • Download and install ngrok from their official website https://ngrok.com/download.
  • After installation, open up its command line.
  • You need to sing up in their website and get a token and give it to the CMD, it's easy you can follow the instructions in the website.
  • Have your website hosted on your localhost, you can use live server in VS code.
  • Then in the ngrok CMD, type the below command, make sure to use the right port.
    • ngrok http localhost:8000
