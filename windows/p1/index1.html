    • | How to transfer files between iPhone and Windows without any wire?



    - There are lots of methods out there but this one doesn’t need installing anything and it uses P2P connection. (I’ll explain P2P on other post but for now just consider that there would be no server in the middle so it’s all on your local network;

    ShareDrop

    1. Go to the https://www.sharedrop.io/ from both iPhone and Windows.
    2. Click on the + (plus sing) on the top right corner.
    3. Scan the QR code or paste the given URL into your iPhone, If the connection is stable, you’ll see two different icons on both pages.
    4. Now you can transfer file from both devices just by clicking on the icon.
