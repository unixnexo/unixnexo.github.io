    • | How to configure a Mikrotik router (using WinBox)



    1. Connect the router with a Lan cable to the computer. (use the ports 2 to 5, do not use the first port since the firewall will block the configuration.
    2. Download WinBox from the official website → https://mikrotik.com/download
    3. Open the the app and hit the the Refresh on the neighbors section


    4. You should see the router down side the page


    5. Double click on the MAC Address tab, it should appear in the Connect To field
    6. The default Username(Login) is admin and leave the Password empty
    7. Hit the Connect button
    8. Now you should see the admin panel


    9. Click on Quick Set. New windows will show up.
    10. Change the Quick Set to Home AP.
    11. On the Internet section, change the Address Acquisition to Automatic. (It depends to your ISP but mostly it’s Automatic).
    12. On the Local Network section, se the IP Address to 192.168.5.1. To prevent confliction with your main router, do not put the IP Address like (192.168.0.1 or 192.168.1.1).
    13. Choose 255.255.255.0(/24) for the Netmask.
    14. Check the DHCP Server.
    15. Enter the range 192.168.5.10-192.168.5.254 in the DHCP Server Range. Also enable NAT. (The reason we are setting the range from 10 is that you may have other devices in the range of 0 to 9).


    16. Click Apply and then OK.


      17. Setting up wireless

    17. On the left-side menu click on Wireless. You should see the wlan1.
    18. Double click on wlan1. You should see Interface tap show up.
    19. Go to the second tap which is Wireless and make sure that Mode is set to ap bridge.
    20. Put the Frequency on auto.
    21. You can set the SSID to anything you wish. (It’s the wireless name).
    22. Then click ok.




      23. Security profiles

    23. On the Wireless section go to the Security Profiles.
    24. Double click on the default. You should see a window.
    25. Change the Mode to dynamic keys and check WPA2 PSK.
    26. Put the wireless password you wish into WPA2 Pre-Shared Key.
    27. Click Apply and then ok.




      28. Rebooting the router

    28. On the left-side menu click on System and choose Reboot.

      29. Now you can:

      Use the first port and also if you connect your internet to your Mikrotik, You’ll have internet connection.



      How to set a password

    29. On the left-side menu go to System and then select Password.
    30. If you haven’t put a password on your router, then just leave the Old Password empty.
    31. Enter your new password twice and click on Change, then restart the app. (re-open it).


      32. Managing users

    32. On the left-side menu go to System and then select Users.
    33. Click on the + sing on the top-left.
    34. Enter the name and select the permission you wish to give the user. (For more understanding go to the Groups section)
    35. Set the user password and click Ok.




      36. Upgrading OS & Firmware

    36. On the left-side menu go to System and then select Packages.
    37. You see some packages that are already installed. Fade ones are disabled.
    38. Click on Check For Updates (Make sure that your router is connected to the internet).
    39. If you click on Download, it will download the packages but won’t install them till you reboot it manually.
    40. If you click on Download and Install, it will download and install the packages by rebooting the router automatically.
    41. It may take a while till you can log in so be patient.


      42. Upgrade the router board

    42. On the left-side menu go to System and them select RouterBOARD.
    43. Click on Upgrade. After that go to System and Reboot.
    44. Give it a minute or so to reboot and log in back.


      45. Installing Packages

    45. Go to https://mikrotik.com/download and download the appropriate packages base on your router package type.
    46. On the left-side menu go to System and then select Packages.
    47. Just drag and drop any packages from the file you’ve already downloaded.
    48. Go to System and then Reboot the router in order to install the packages.


      49. Configuring using web (WebFig)

      - You can also configure the router using web interface.

    49. Enter the IP we have already given to the router in browser which is 192.168.5.1.
    50. If you haven’t set the IP address, the default is 192.168.88.1.
    51. Log in and go to the WebFig tab or just follow along the quick set.
